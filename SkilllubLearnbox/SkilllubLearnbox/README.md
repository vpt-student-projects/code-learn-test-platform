LearnBox - это полнофункциональная образовательная платформа для онлайн-обучения. Бэкенд написан на ASP.NET и предоставляет REST API для управления пользователями, курсами, уроками и тестами. Основные функции включают регистрацию и аутентификацию пользователей с JWT токенами, ролевую систему с тремя уровнями доступа (студент, преподаватель, администратор), управление образовательным контентом через курсы, модули и уроки, систему тестирования с вопросами и вариантами ответов, восстановление пароля через email с шестизначными кодами подтверждения, и админ-панель для управления пользователями и просмотра статистики.

Платформа использует современные веб-технологии: бэкенд построен на ASP.NET Core с Entity Framework для работы с базой данных PostgreSQL через Supabase, аутентификация реализована через JWT токены с автоматическим обновлением. Система обеспечивает безопасность через хеширование паролей с BCrypt, валидацию входных данных и защиту API endpoints по ролям.
